

PRD: Landing Page Sistem Pengelolaan Air Desa (Smart-Air Desa)
1. Ringkasan Proyek
Membangun landing page profesional untuk aplikasi sistem pengelolaan air berbasis web guna meningkatkan transparansi, efisiensi penagihan, dan kemudahan akses informasi bagi warga desa.
Visi: Menjadi pusat informasi digital untuk layanan air desa yang kredibel dan modern.
Target Audiens: Warga desa (pelanggan), Pengelola BUMDes (admin), dan Perangkat Desa.
2. Branding & Identitas Visual
Aplikasi ini akan menggunakan palet warna yang kontras namun harmonis untuk memberikan kesan segar (air) dan hangat (pelayanan masyarakat).

Struktur Konten Landing Page
A. Header (Navigasi)
Logo: Nama Sistem (contoh: TirtaDesa Digital).
Menu: Beranda, Fitur, Cek Tagihan, Cara Bayar, Pengaduan.
Tombol Login: Akses ke dashboard admin/pelanggan (Warna: Light Sea Green).
B. Hero Section (Utama)
Headline: "Kelola Air Desa Lebih Praktis, Transparan, dan Modern."
Sub-headline: "Pantau penggunaan air, cek tagihan, dan ajukan keluhan hanya dalam satu genggaman."
CTA Utama: [Cek Tagihan Sekarang] (Warna: Amber Glow).
Visual: Ilustrasi atau mockup dashboard aplikasi web di perangkat mobile/laptop.
C. Statistik Cepat (Social Proof)
Menampilkan angka real-time untuk membangun kepercayaan:
Total Pelanggan Aktif.
Total Debit Air Tersalurkan (mÂ³).
Jumlah Dusun Terlayani.
D. Fitur Utama (Core Features)
Disusun dalam bentuk card dengan ikon modern:
Monitoring Pemakaian: Lihat histori pemakaian bulanan secara detail.
E-Billing: Tagihan otomatis terkirim via WhatsApp/Email.
Pembayaran Online: Integrasi dengan bank atau e-wallet (Opsional).
Layanan Pengaduan: Form digital jika ada kebocoran atau kendala distribusi.
E. Panduan Pengguna (How It Works)
Langkah mudah menggunakan sistem dalam 3 tahap:
Input ID Pelanggan.
Lihat Detail Pemakaian.
Lakukan Pembayaran.
F. Footer
Informasi Kontak (Kantor Desa/BUMDes).
Media Sosial Desa.
Copyright & Link Kebijakan Privasi.
4. Persyaratan Fungsional
Cek Tagihan Cepat: Fitur input ID Pelanggan langsung di landing page tanpa perlu login terlebih dahulu.
Responsive Design: Tampilan harus sempurna di HP (Android/iOS) karena mayoritas warga mengakses via smartphone.
Fast Loading: Optimasi gambar agar website ringan di akses di area dengan sinyal terbatas.
5. Persyaratan Teknis (Rekomendasi)
Framework: vue + tailwind v3 + meraki UI.
Typography: Menggunakan font plus jakarta sans agar terlihat modern dan mudah dibaca.
Animasi: Gunakan AOS (Animate On Scroll) ringan pada bagian card fitur.
6. Contoh Implementasi Warna pada Elemen
Background: Gunakan #CBF3F0 (Frozen Water) sebagai warna dasar section agar mata terasa sejuk.
Tombol Aksi: Gunakan #FF9F1C (Amber Glow) untuk tombol "Bayar Sekarang" atau "Lapor Bocor" agar terlihat mencolok di antara warna hijau/biru.
Teks & Ikon: Gunakan #2EC4B6 (Light Sea Green) untuk judul fitur agar identitas "air" tetap kuat.
